# Kalyanam - Premium Matrimony Platform

A modern, scalable, and beautifully designed Indian Matrimony Web Application frontend built with React (Vite), Tailwind CSS, and React Router.

## Architecture & Tech Stack
- **Framework**: React 18 (via Vite for lightning-fast HMR and building)
- **Routing**: React Router DOM v6
- **Styling**: Tailwind CSS v3 with a custom *Premium Maroon, Gold & Cream* theme
- **State Management**: React Context API (`AuthContext` for JWT tokens)
- **API Client**: Axios (with custom interceptors for JWT injection)
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## Features
- **Public Flow**: Fully responsive Landing Page, Registration (with multi-step feel), OTP Verification, and Login.
- **Protected Routes**: Dashboard, Match Explorer, Profile Update, and Profile Detail views.
- **Premium UI/UX**: Soft shadows, warm color palette, micro-interactions, skeleton loaders, and modern layouts.
- **Form Handling**: Massive multi-category profile update form structured into beautiful cards.

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Variables**
   Create a `.env` file in the root directory and add your configurations.
   ```bash
   cp .env.example .env
   ```
   *By default, the API is configured to point to `http://localhost:5000/api/users`.*

3. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Navigate to `http://localhost:5173/` in your browser.

4. **Build for Production**
   ```bash
   npm run build
   ```

## Design System Variables
The application uses a tailor-made design system configured in `tailwind.config.js`:
- **Primary**: Maroon (`#8f2c2c` base)
- **Secondary**: Gold (`#e0b215` base)
- **Neutral**: Cream (`#fdfbf7` base)
- **Fonts**: *Inter* (sans-serif) & *Merriweather* (serif headings)
- **Shadows**: Custom `soft` and colored `premium` drop-shadows.

## Folder Structure
- `src/components/` - Reusable UI widgets (Navbar, Footer)
- `src/pages/` - Core application views
- `src/context/` - Global state (AuthContext)
- `src/services/` - Axios wrapper and mock integrations
- `src/routes/` - Route index and `ProtectedRoute` definitions
